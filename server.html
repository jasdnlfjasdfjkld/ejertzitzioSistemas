SERVER

-Los procesadores de los Servidores son multiprocesadores
-Utilizan un socket especial
-Estan a todo momento en funcionamiento, son capaces de estar mucho tiempo sin tener que apagarse.
-Estos procesadores no se utilizan para reproducir gráficos, vienen sin gráficos integrados.

LAPTOP

-Los procesadores de portatiles tienen un espacio reducido, esto hace limitar el tamaño de los componentes.
-Consumen menos energía, el TDP es la mitad que del sobremesa.
-Utilizan el socket bga, la cpu va soldado a la Placa.
-Tienen menos nucleos, sin hyperthreading / SMT para doblar el num de hilos
-Tienen menos velocidad de proceamiento GHz (por eso consumen menos energia)
-Casi todos llevan graficos integrados
-La misma Gpu dedicada tiene menos rendimiento en portatiles que en sobremesas. Se puede usar una EGPU (gpu externa) conectada via USB / thunderbolt, pero pierde rendimiento, transferencia de datos... Hoy en día los gráficos integradas son mejores porque van conectados vía PCIe.
